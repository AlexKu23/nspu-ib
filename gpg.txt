Процедура генерации ключей для шифрования, зашифровки и расшифровки файлов 
1. Генерация ключей
gpg --full-generate-key
Отвечаем на все вопросы (выбираем алгоритм, пригодный для шифрования),
 указываем срок действия ключа, длину, свои данные и парольную фразу

2. Просмотр ключей, уствновленных в списке 
gpg --list-key

3.Экспортируем открытый ключ в файл:
 gpg --export alex23ku@ya.ru > <имя файла>

4.Отправляем полученный файл с открытым кодом пользователю, который должен прислать зашифрованное сообщение
Поьзователь  шифрует свой файл при помощи нашего открытого ключа

5. Импорт открытого ключа в систему пользователя
gpg --import <имя файла>
Просмотр: gpg --list-key

6.Процесс шифрования файла открытым ключом:
gpg --out <имя входного фала> --recipient alex23ku@ya.ru --encrypt <имя входного фала>
Пользователь направляет нам зашифрованный фал для дешифровки

7.Процесс расшифровки файла закрытым ключом:
gpg --out <имя входного файла> --decrypt <имя зашифрованого файла>
Вводим парольную фразу, сосоставленную секретному ключу. 
Просматриваем зашифрованный файл
